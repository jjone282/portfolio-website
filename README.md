# Jaclyn Jones — Portfolio Website

Personal portfolio website built from scratch with vanilla HTML, CSS, and JavaScript.

## Live Site

[jjone282.github.io/portfolio-website](https://jjone282.github.io/portfolio-website)

## Features

- **Hero section** with animated code snippet typewriter effect and custom cursor
- **About** section with photo and resume download
- **Education** section with GPA, graduation date, and coursework
- **Skills** grid with pink/lavender diagonal accent pattern
- **Experience** timeline with roles at Cable One, ASU, and the Carson Athletic Center
- **Projects** horizontal scroll with links to GitHub repos
- **Contact** form with social links
- Scroll-triggered fade-in animations
- Fully responsive — adapts to mobile, tablet, and desktop
- Custom cursor with lagged ring follow (desktop only)
- Mesh gradient hero background

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla — no frameworks)
- Google Fonts: [Lora](https://fonts.google.com/specimen/Lora) + [Nunito](https://fonts.google.com/specimen/Nunito)
- Hosted on GitHub Pages
